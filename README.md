# How to Play
When you first start the game it will ask for your save data, if it is your first time playing just hit enter and it will start you from nothing, if you do have save data paste it into the input line and it will do the work for you, if you want to buy something hit "ctrl + C" to keyboard interupt the program then it will ask what you want to buy, there is an endless number of upgrades however they quite costly at high levels format the upgrades you wish to buy as such: "'upgrade you want to buy' 'number of said upgrades'" then hit enter, to quite the game hit ctrl + c twice and copy your save data.
